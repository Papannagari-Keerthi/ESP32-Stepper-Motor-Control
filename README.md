# ESP32 Stepper Motor Control

## Overview

This project demonstrates stepper motor control using an ESP32 microcontroller and the AccelStepper library. The motor speed is controlled through analog input.

## Components Used

* ESP32
* Stepper Motor
* Motor Driver
* Potentiometer / Analog Input

## Features

* Stepper motor interfacing
* Speed control using analog input
* Real-time motor operation
* Embedded systems implementation

## Working Principle

The ESP32 reads analog input values and adjusts the stepper motor speed accordingly using the AccelStepper library. The motor runs continuously at the configured speed.

## Technologies Used

* ESP32
* Arduino Programming
* AccelStepper Library
* Embedded Systems
